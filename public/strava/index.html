      What is this?
      

        You just authorized fitness-tools' read access to your Strava activities. This page captured
        the one-time authorization code from the redirect URL and packaged it into a JSON payload
        with the granted scopes and a timestamp. Copy the payload and paste to Aria so she can
        exchange it for an access token. The code is single-use and expires in minutes — there's
        no value in storing it elsewhere.
