      What is this?
      

        You just authorized stockpot's read-only access to your Schwab account. This page captured
        the one-time authorization code from the redirect URL and packaged it into a JSON payload
        with the session token and a timestamp. Copy the payload and paste to Aria so she can
        exchange it for an access token. The code is single-use and expires in a few minutes.
